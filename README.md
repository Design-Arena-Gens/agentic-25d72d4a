# De Jongh’s Panelbeating Centre – AI Assistant

Next.js app providing a friendly, bilingual assistant for services, estimates, bookings, job status, and after-care tips.

- Stack: Next.js 14, React 18, TypeScript
- Deploy: Vercel

## Development

- Install: `npm install`
- Run: `npm run dev`
- Build: `npm run build`
- Start: `npm start`
